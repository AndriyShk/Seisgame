# Seismic: Core Dodger 🌋

**Seismic: Core Dodger** is an adrenaline-fueled browser game where you fall deep into the Earth's layers. Dodge rocks, boulders, and lava pools as you race to reach the red-hot core at 5000m!

## 🎮 How to Play

- **Desktop**: Use `Arrow Left` / `Arrow Right` or `A` / `D` keys to move.
- **Mobile**: Use the on-screen buttons or your device's tilt sensor.
- **Goal**: Reach **5000m** depth without crashing. Your speed increases as you go deeper!

## ✨ Key Features

- **Dynamic Difficulty**: Obstacles spawn faster and speed increases the deeper you go.
- **Atmospheric Visuals**: Parallax backgrounds and 3D tunnel shadows simulate depth transitions from the Crust to the Inner Core.
- **Endgame Sequence**: A cinematic transition to a "void" before entering the planetary core.
- **Responsive Controls**: Precise movement for both keyboard and mobile devices.
- **Social Sharing**: Share your high records directly on **X (Twitter)**.
- **High Contrast UI**: Sharp graphics and neon-glow effects for a premium arcade feel.

## 🛠️ Tech Stack

- **Core**: Vanilla JavaScript (ES6+), HTML5 Canvas.
- **Styling**: Vanilla CSS3 (Design Token Architecture).
- **Icons**: SVG-based dynamic favicons.

## 🚀 Local Deployment

1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. For the best experience, host it using a local server (e.g., VS Code Live Server).
